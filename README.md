# Интернет-магазин электроники
* Стек: Java 17, Spring Framework, Jpa, PostgreSQL, HTML, CSS
* Паттерн: MVC

* БД - PostgreSQL, 
* Для взаимодействия с сущностями БД - JPA
* Веб-страницы - HTML
* Стили веб-страниц - CSS
